# Homebridge OJ Electronics Plugin

This plugin adds support for OJ Electronics (and re-branded) underfloor heating thermostat controllers to homebridge

Currently supports devices branded as:
- OJ Electronics
- Heatmat
- Schlüter “Ditra-Heat”
